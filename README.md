# Cappl Screen Recorder Extension

A professional Chrome extension for high-quality screen recording with advanced annotation and editing capabilities.

## Features

### Core Recording Features
- **Screen Recording**: Record your entire screen, a specific tab, or a selected area
- **Audio Recording**: Capture system audio and/or microphone input
- **Multiple Qualities**: Choose between 720p, 1080p, or 4K resolution
- **Auto Download**: Option to automatically download recordings after completion

### Real-Time Annotation Tools
- **Pen Tool**: Draw with adjustable color and thickness
- **Eraser**: Remove annotations
- **Highlighter**: Highlight important areas of the screen
- **Undo/Redo**: Unlimited undo/redo history for all annotations
- **Clear All**: Remove all annotations at once

### Drawing Bar Settings
- **Master Toggle**: Enable/disable all drawing tools with a single switch
- **Tool Toggles**: Individually enable/disable each tool (pen, highlighter, etc.)
- **Color Palette**: Customize colors for each tool
- **Thickness Slider**: Adjust line thickness for pen and highlighter
- **Transparency Control**: Set opacity levels for annotations
- **Keyboard Shortcuts**: Quick access to all drawing tools

### Recording Settings
- **Audio Options**: Enable/disable microphone recording
- **Video Quality**: Select recording resolution (720p, 1080p, 4K, Auto)
- **Auto Hide**: Option to hide the recording interface during recording
- **Countdown**: Enable/disable recording countdown timer (3, 5, or 10 seconds)

### Browser Integration
- **Start Recording**: Capture current tab, entire screen, or selected area
- **Stop Recording**: Stop recording and save the video
- **Recording Preview**: View and play recordings in browser tabs
- **Quick Access**: Access recording controls via browser action button

## Usage

### Starting a Recording
1. Click the Cappl icon in your browser toolbar
2. Select "Start Recording"
3. Choose what to record: Current Tab, Entire Screen, or Selected Area
4. Configure audio settings and recording quality
5. Click "Start Recording" to begin

### During Recording
- The recording interface will appear with the drawing tools
- Use the tools to annotate your screen in real-time
- Click "Stop" when you're finished
- The recording will be saved to your computer

### Settings
To customize your recording experience:
1. Click the gear icon in the top-right corner of the recording interface
2. Select "Settings"
3. Configure drawing bar options, recording settings, and shortcuts
4. Changes are saved automatically

### Managing Recordings
1. Open the recording interface
2. Go to the "Recordings" tab
3. View your saved recordings
4. Play, download, or delete recordings

## Keyboard Shortcuts

### Recording Controls
- `Ctrl + M`: Start/Stop Recording (Windows/Linux)
- `Cmd + M`: Start/Stop Recording (Mac)
- `Ctrl + Shift + S`: Stop Recording (Windows/Linux)
- `Cmd + Shift + S`: Stop Recording (Mac)

### Drawing Tools (When Recording Interface is Open)
- `D`: Toggle Pen Tool
- `H`: Toggle Highlighter Tool
- `E`: Toggle Eraser Tool
- `C`: Clear All Annotations
- `Ctrl + Z`: Undo
- `Ctrl + Shift + Z`: Redo
- `1`: Select 1st Color
- `2`: Select 2nd Color
- `3`: Select 3rd Color

## Technical Information

### Supported Formats
- **Video**: WebM (VP9 codec)
- **Audio**: Opus

### Codec Support
- **Video**: `video/webm;codecs=vp9` (recommended)
- **Audio**: `audio/webm;codecs=opus` (recommended)

### Browser Compatibility
- **Chrome**: Fully supported
- **Firefox**: Experimental support
- **Edge**: Fully supported
- **Safari**: Not supported

### Screen Recording Options
- **Current Tab**: Captures only the content of the active tab
- **Entire Screen**: Captures the entire desktop display
- **Selected Area**: Requires user to select a specific region to record
